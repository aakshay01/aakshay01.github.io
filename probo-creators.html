User-Generated Events Platform — Scale Content, Reduce Ops

Probo 2021 - 2025 Product Lead
← Back to Portfolio

Overview

Creators was a program that allowed power users to create events for the Probo platform. Their name appeared on the event card, giving them recognition. This diversified our event catalog while significantly reducing internal operations effort.

The Challenge: Internal ops couldn't keep up with event demand across all categories. We needed a scalable way to diversify content without compromising quality.

My Role

  • Designed the creator onboarding and verification flow
  • Built the event creation tools for creators
  • Created the attribution system (name on event card)
  • Developed quality guidelines and review processes
  • Managed creator relationships and feedback loops

Key Features

  • Creator Attribution: Creator's name featured on event cards — recognition and accountability
  • Event Creation Tools: Streamlined interface for creating prediction events with proper rules
  • Quality Review: Lightweight review process to maintain platform standards
  • Creator Dashboard: Stats on event performance, trader engagement

Learnings

Creators taught me the power of community-generated content. By giving users ownership (their name on events), we created intrinsic motivation beyond monetary rewards. The key insight: attribution drives quality because creators have skin in the game.
